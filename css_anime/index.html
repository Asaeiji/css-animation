概要

普段、ゲームサイトやアニメサイトを見ているときにアニメーションがきれいなサイトを見て自分でも作って見たいと思い今回cssフレームワークを作ろうと思いました。

このフレームワークの名前も簡単にですが作ってみました。

c: css
a: アニメーション
f: フレームワーク

説明

このフレームワークはbootstrapのようなレイアウトの枠組みではなく主にアニメーションに関してのフレームワークを作りました。
アニメーションの数は正直まだまだ少ないです。

使い方

こちたの使い方は、css/anime.cssを読み込んで頂いてコンポーネントを見ていただき好きなアニメーションを適用してください。
